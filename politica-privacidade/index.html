Política de Privacidade — Cataliad Auto DM

Última atualização: 17 de junho de 2026

Este serviço ("Cataliad Auto DM") automatiza respostas por mensagem direta (DM) no Instagram quando seguidores comentam palavras-chave nas publicações da conta profissional @cataliad.dev. Esta política descreve quais dados tratamos e como.

1. Dados que coletamos

  • Comentários públicos nas publicações da conta: texto do comentário, identificador e nome de usuário de quem comentou.
  • Identificador do remetente da DM, necessário para enviar a resposta automática.
  • Registro de envios (qual DM foi enviada e quando), para respeitar limites de frequência (1 DM por usuário a cada 24h).

2. Como usamos os dados

Os dados são usados exclusivamente para identificar a palavra-chave no comentário e enviar a mensagem direta correspondente. Quando configurado, o texto do comentário pode ser enviado a um modelo de linguagem (Anthropic Claude) apenas para gerar uma resposta personalizada. Não vendemos nem compartilhamos dados com terceiros para publicidade.

3. Armazenamento e retenção

As regras de automação e o registro de envios são armazenados em banco de dados (Supabase). Os dados são mantidos apenas enquanto necessários para a operação do serviço e podem ser excluídos a qualquer momento.

4. Compartilhamento com terceiros

Utilizamos: a API do Instagram (Meta) para receber comentários e enviar DMs; o Supabase para armazenamento; e a Anthropic (Claude) para gerar respostas personalizadas (quando ativado). Cada um trata os dados conforme suas próprias políticas de privacidade.

5. Seus direitos

Você pode solicitar a exclusão dos seus dados ou parar de receber mensagens a qualquer momento, entrando em contato pelo e-mail abaixo. Atenderemos a solicitação no menor prazo possível.

6. Exclusão de dados

Para solicitar a remoção de todos os seus dados do nosso sistema, envie um e-mail para rr2trafego@gmail.com com o assunto "Exclusão de dados" e o seu nome de usuário do Instagram. Os dados associados serão apagados.
